# Max-Cut NP-Complete Visualizer

An interactive step-by-step visualization tool for the Max-Cut problem, built entirely with vanilla HTML5, CSS3, JavaScript, and SVG — no external libraries or frameworks.

🔗 Live Demo: https://ahmedalabsi-cyber.github.io/max-cut-visualizer/

---

## Features

- Graph Builder — add custom edges or load preset graphs
- Brute Force Solver — tries all possible 2ⁿ partitions
- Greedy Local Search — flips vertices to improve the cut
- Side-by-side Comparison — compares brute force and greedy performance
- Pseudocode Panel — active line highlighting during execution
- Step Explanation — explains each algorithm step clearly
- NP-Completeness Page — shows the proof structure and NAE-3-SAT reduction
- Growth Chart — compares exponential vs practical algorithm growth

---

## Algorithm

- Exact approach: Brute Force
- Practical approach: Greedy Local Search
- Approximation guarantee: 0.5 approximation
- Brute force complexity: O(2ⁿ · |E|)
- Greedy local search complexity: O(|V| · |E|)
